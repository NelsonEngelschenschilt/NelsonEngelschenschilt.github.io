Physically Based Rendering 3D Art

Shark Raw Soyouz Helmet by Nelson Engelschenschilt on Sketchfab

Project Description

To achieve this result I made high poly models using 3ds Max turbosmooth. Then I bake the normal and ambient occlusion map onto a low poly version of the model and create the diffuse and metalness textures using photoshop.

Created Using

  • Unreal Engine 4
  • Photoshop
  • xNormal
  • 3ds Max
