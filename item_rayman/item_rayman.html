Rayman Advance Clone Gameplay programming

Project Description

Rayman is a treasured childhood memory of mine that introduced me to the magical world of games. Therefore I decided to make a Rayman clone as my very first game project during which discovered the basic concepts behind gameplay programming. I also experimented with more advanced concepts such as parallax backgrounds and sprite animations.

Created Using

  • C++
  • Inkscape
  • Howest game engine
  • Box2D physics engine
